# TR2EN - Türkçe-İngilizce Kelime Veritabanı

TR2EN, Türkçe kelimelerin İngilizce karşılıklarını içeren kapsamlı ve düzenli bir veri kümesidir.

TR2EN is a comprehensive and organized dataset containing English equivalents of Turkish words.

## Özellikler (Features)
- **Geniş Kelime Dağarcığı:** On binlerce kelime çifti.
- **Kolay Entegrasyon:** JSON formatında veri erişimi.
- **Açık Kaynak:** Herkesin katkısına açık.
